Aprenda a Estruturar um Código HTML

Entenda o básico de HTML de forma simples e prática!

1. Estrutura Básica

Todo arquivo HTML segue uma estrutura padrão que garante que ele seja interpretado corretamente pelo navegador.

Eis como começa:

!DOCTYPE html
html lang="pt-BR"
    head
       !-- Configurações da página --
    /head
    body
        !-- Conteúdo visível --
    /body
/html

2. Cabeçalho (head)

O cabeçalho não mostra nada na página, mas é essencial para definir coisas como o título da aba, o estilo e informações importantes sobre a página.

Exemplo de um cabeçalho:

head
    meta charset="UTF-8"; 
    meta name="viewport" content="width=device-width, initial-scale=1.0";Adapta para mobile -->
    titleMinha Primeira Página/title 
    link rel="stylesheet" href="styles.css" 
/head

3. O Corpo (body)

É aqui que fica o conteúdo visível da página: textos, imagens, botões, links e tudo que o usuário interage.

Veja um exemplo simples:

body
    h1 Bem-vindo!/h1 
    p Essa é a minha página de exemplo./p 
/body

Tags mais utilizadas em HTML

Veja abaixo uma lista das tags HTML mais comuns e suas funções:

  • html Define o documento HTML.
  • head Contém informações sobre o documento, como metadados e links para estilos.
  • title Define o título que aparece na aba do navegador.
  • body Contém todo o conteúdo visível da página.
  • <h1; a; h6 Usadas para títulos, com diferentes níveis de importância.
  • p Define um parágrafo.
  • a Cria links para outras páginas ou seções.
  • img Insere imagens.
  • ul; Cria listas desordenadas (com marcadores).
  • ol Cria listas ordenadas (numeradas).
  • li Define itens dentro de listas.
  • div Cria divisões de conteúdo, útil para organizar o layout.
  • span Define pequenos trechos de texto para estilização específica.
